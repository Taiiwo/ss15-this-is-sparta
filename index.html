How it works?

Pear Programming is made for developers who like to share their experience with others by pairing with them.

Here's how you can use it:

  1. Create or join a room
  2. Wait for users to join you
  3. Start coding!

Awesome right? but you must follow these rules...

  1. There's a maximum of 5 users for each room
  2. You must must use HTML + CSS + Javascript (we're looking foward to implement more options)
  3. If the room is full, you can WATCH the room (no coding neither sharing your sound and video)
